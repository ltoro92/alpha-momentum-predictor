# Experimento v0 — Alpha Momentum Predictor

Estado: congelado para validación inicial  
Versión: 2.1  
Capital ficticio inicial: 200 USDT  
Duración: 7 días calendario  
Mercado: Binance Alpha  

## Propósito

Evaluar si existe una ventaja estadística medible en la detección temprana de aceleraciones especulativas en tokens de Binance Alpha usando reglas simples, objetivas y reproducibles.

El experimento usará datos reales de mercado, pero capital ficticio.

## Hipótesis inicial

Los tokens de Binance Alpha con variación 24h moderada, volumen relativo creciente, momentum temprano y liquidez suficiente tienen mayor probabilidad de alcanzar +20%, +35% o +50% dentro de las siguientes 24 horas que estrategias base como selección aleatoria, momentum simple, compra tardía de top gainers o filtro básico sin scoring.

## Objetivo financiero experimental

- Meta fuerte: 600 USDT
- Meta de victoria: 1000 USDT

Estas metas no validan por sí solas el sistema. La validación técnica depende de EV positivo, profit factor, drawdown, slippage, liquidez y benchmarks.

## Principios

- Si no se puede medir, no existe.
- Si no se puede ejecutar, no cuenta.
- Si no considera pérdidas, no es real.
- Si no supera benchmarks, no hay edge.
- Si depende de suerte extrema, no es sistema.
